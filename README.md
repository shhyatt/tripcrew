# TripCrew

TripCrew is a group travel planning app that lets friends collaborate on trips, share ideas, and build itineraries together.

## ✈️ Features (MVP)

- Create and manage trips
- Add friends to a trip
- Suggest activities (trip ideas)
- Build a shared itinerary
- Track trip prep with a checklist

## 🧱 Database

This project uses PostgreSQL.
